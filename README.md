# DEM
An implement of the algorithm to build Delaunay Triangulation.
## Usage
use cpp to build the Triangulation.  
`gcc main.cpp`  
for windows, use  
`main.exe`  
or  
for linux, use  
`./main.o`  
then, use python to draw the delaunay triangulation  
`python draw.py`
